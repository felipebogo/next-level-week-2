# NEXT LEVEL WEEK #2

<p>Aplicação web para cadastro de aulas desenvolvida na segunda edição do evento Next Level Week.</p>
<p>O código backend está disponível na pasta server e o código frontend está na pasta web.</p>

## Backend
### Install:
```bash
cd server
yarn
```
Executar migrations
```bash
yarn knex:migrate
```

### Execute:
```bash
yarn start
```
<p>O backend irá executar na porta 3333.</p>

## Frontend
### Install:
```bash
cd web
yarn
```

### Execute:
```bash
yarn start
```
